cd

#BRANCHING WORKFLOW#
1.Create a new branch.
git checkout -b branch_name

2.List all branches
git branch

3.Switch Branches (from one branch to another branch)
git checkout main
git checkout feature-login

4.Make Changes
FLOW :- Add -> Commit -> Push

	4.1 Check status
	git status

	4.2 Stage Files
	git add file_name

	to reflect all changes in file
	git add .

	4.3 Commmiting changes
	git commit -m "Message"

	4.4 Push changes to GitHub
	git push origin main/branch_name

	Push changes into the current part
	git push
